# Wait For Window To Exist Command


## What does this command do?
This command waits for a window to exist.


## When would I want to use this command?
Use this command when you want to explicitly wait for a window to exist before continuing script execution.


## Command Parameters
| Parameter Question   	| What to input  	|  Sample Data 	| Remarks  	|
| ---                    | ---               | ---           | ---       |
|Please enter or select the window name that you are waiting for to exist.|Input or Type the name of the window that you want to wait to exist.|**Untitled - Notepad**||
|Indicate how many seconds to wait before an error should be raised.|Specify how many seconds to wait before an error should be invoked|**5**||
